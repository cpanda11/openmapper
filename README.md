## Usage

Node.js must be installed on the system.

### Install
```sh
cd client

$ npm install
```

### Start Local Instance
```sh
$ npm run start
```

### Run Tests
```sh
$ npm run test
```


### Build
```sh
$ npm run build
```

Open browser and type: localhost:8080
